# ThirdPersonNametag

A simple mod to show your own nametag in third person view for Minecraft Bedrock Edition (Windows).

## Features

- Shows your nametag when in third person view.
- Supports v1.21.132.01

## Requirements

- [LeviLauncher (GDK)](https://github.com/LiteLDev/LeviLauncher)

## Installation

1. Download `ThirdPersonNametag.dll` from [Releases](https://github.com/RadiantByte/LeviLauncher-ThirdPersonNametag/releases) or GitHub Actions.
2. Install LeviLauncher.
3. Add the `ThirdPersonNametag.dll` mod in LeviLauncher.
4. Launch Minecraft through LeviLauncher.
